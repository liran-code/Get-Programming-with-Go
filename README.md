## 该书简介

本书是一本面向 Go 语言初学者的书，循序渐进地介绍了使用 Go 语言所必须的知识，展示了非常多生动有趣的例子，并通过大量练习来加深读者对书中所描述内容的理解。

本书适合对初学 Go 语言有不同需求的程序员阅读。无论是刚开始学习 Go 语言的新手，还是想要回顾 Go 语言基础知识的 Go 语言使用者，只要是想要 Go 做开发，无论是开发小型脚本还是大型程序，本书都会非常有帮助。
